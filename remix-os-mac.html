How to Get Files off a Broken Phone

Required Files:

 

1. ADB and Fastboot Available here: https://goo.gl/WQ1nnG

2. A TWRP image for your device: https://twrp.me/Devices/

 

A Mac Version of this tutorial can be found in the "fastboot" folder

 

Having Trouble?

Contact us using relevant page.

 

 

This tutorial is Designed for Windows a macOS version can be found in the fastboot folder

 

 1. Enter Boot loader or Download Mode

(On many devices this is accessed by pressing and holding Power + Volume Down for around 4-6 seconds depending on your device.)

 

2.  Connect your device to the computer. Check that your phone is connected by running: fastboot devices

If your device is not listed check the cable is connected correctly or try another USB cable

 

3. Find the TWRP image that you downloaded before and copy it to the "fastboot" folder, then rename the file to recovery.img

 

4. Open Terminal in the "fastboot" folder by opening the folder, then holding Shift and Right clicking selecting "Open Terminal Window here"

 

(The next step will allow you to enter recovery mode. The second command will boot your device straight into the recovery image.

This is recommended.

The second will wipe over the existing command and you will need to manually boot into recovery mode. Try this if the second method fails.)

 

5. From here assuming you have renamed the recovery file to recovery.img enter the following command:

fastboot flash recovery recovery.img

(This will delete the stock recovery)

or

fastboot boot recovery.img

(This will keep the original recovery and boot straight to the file.)

 

6. If you ran the first command to wipe the stock recovery, enter recovery mode.

If you used the second command you will automatically boot into the recovery.

 

7. Your PC should detect your phone as a media device and the files will be displayed
